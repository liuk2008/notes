Web语言：Html5 + Css + JavaScript

前端框架：React.js、Vue.js、Angular.js 
混合框架：React-Native、Weex

RN环境搭建：
1、JDK、Android、Git、Python、Node.js
2、设置npm镜像
	npm config set registry https://registry.npm.taobao.org --global
	npm config set disturl https://npm.taobao.org/dist --global

C:\Users\Administrator\AppData\Roaming\npm\etc\npmrc

registry=https://registry.npm.taobao.org/
disturl=https://npm.taobao.org/dist

3、安装react-native命令行工具
	npm install -g react-native-cli
3、在工作目录中创建项目，
	运行	react-native init MyProject
4、运行packager
	react-native start
